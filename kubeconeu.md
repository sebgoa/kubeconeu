## kubeless

* Kubernetes extension
* CRD for `functions`
* A controller (acutally several)
* Creates deployments, services
* Creates Ingress if needed
* Uses Configmap, Init container for now
* Instrumented runtimes with `prometheus-client`
* A cute UI
* A `serverless` Plugin
* Custom metrics HPA
